## 📖 Introduction
I hate writing assignments by hand, so I built this tool! It converts your digital text into an image or PDF that looks exactly like human handwriting. Perfect for "handwritten" submissions in a digital age. 😛

## ✨ Features
- **Real-time Conversion:** See your text transform as you type.
- **Custom Fonts:** Realistic handwriting styles.
- **Export Options:** Download your work as high-quality Images or PDFs.
- **Paper Styles:** Options for ruled or plain paper backgrounds.

## 🛠️ Built With
- **[html2canvas](https://github.com/niklasvh/html2canvas)** - To render the DOM into a canvas.
- **[jsPDF](https://github.com/MrRio/jsPDF)** - For seamless PDF generation.
- **[Cypress](https://github.com/cypress-io/cypress)** - For end-to-end testing.
- **[Serve](https://github.com/zeit/serve)** - For local development.

## 🚀 How to use
1. Clone the repo: `git clone https://github.com`
2. Open `index.html` in your browser or run `npm start`.
3. Type your content and download your "handwritten" assignment!
